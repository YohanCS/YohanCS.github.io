Discord Bot

!joinedAt to see the date a user has joined said server

BitPic

        

BitPic is designed to allow people to take a photo while traveling and to send it to others in the form of a Bitmoji sticker using their avatar with a simplistic UI
Created in 36 Hours at SB Hacks 2019
Mobile App created in Java using Android Studio Utilizes Google Cloud Platform, mainly uses Cloud Vision API to process images and query for objects.
Also makes use of SnapKit: LoginKit to connect user to the app, BitmojiKit to use the Bitmoji sticker UI, and CreativeKit to be able to click on a bitmoji and send it to another.
Premise is to take a photo or choose from a gallery, process the photo, then show popular word choices based on what the Vision API believes to be in the picture, and then the user taps on a word and it will search through all Bitmoji stickers and return relevant stickers that can be then shared to other apps.
